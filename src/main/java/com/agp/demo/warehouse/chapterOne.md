###自然演化式体系结构
####数据可信性
数据经过多次抽取，然后对抽取数据再抽取，数据源和数据选取粒度和时间度不一样
导致数据缺乏可信度
* 数据无时间基准
* 数据算法上面的差异
* 抽取的多层次问题
* 外部数据问题
* 无公共起始数据源
####生产效率问题
比如报表生成
* 找到需要的源数据
* 编辑数据
* 分析生成报表

自然演化体系，数据源不清晰，各个抽取层繁杂无序，找到合适的粒度表不容易。
本次的报表，来年的时候可能数据源又有变动，又要重费资源完成。
####无法将数据转化为信息
* 各个应用的数据分散，从未集成，进行组合报表时不能完成数据采集完整
* 各个应用历史数据存储的时间长短不一致，不能集合统一数据时间量度
